# Normal mode
Rama para practicar tus habilidades como programador de jr a sr esta rama se enfoca mas a la manera general de programar y en mejorar la logica de programacion

Por supuesto, aquí te proporciono una descripción detallada de cada problema y las restricciones para cada nivel de entrevista técnica.

- **Descripción:** Resuelve el problema de la mochila utilizando programación dinámica. Dado un conjunto de elementos, cada uno con un peso y un valor, determina la cantidad máxima de valor que puedes llevar en una mochila de capacidad limitada.
- **Restricciones:**
  - Puedes usar cualquier lenguaje de programación.
  - Evita el uso de funciones incorporadas para resolver el problema de la mochila.

**Problema 3: Árbol de expresión**

:** Implementa un árbol de expresión y una función que evalúe su resultado. El árbol debe ser capaz de manejar operaciones aritméticas básicas como suma, resta, multiplicación y división.

#### 3. **Entrevista 3:**

**Problema 2: Cifrado RSA**

**Problema 3: Algoritmo de Floyd-Warshall**

### Entrevistas para Desarrollador Senior:

#### 1. **Entrevista 1:**

**Problema 1: Optimización de consulta de base de datos**
- **Descripción:** Diseña una consulta SQL eficiente para recuperar datos específicos de múltiples tablas relacionadas. Optimiza la consulta para minimizar el tiempo de ejecución y el uso de recursos del servidor.
- **Restricciones:**
  - Utiliza SQL para escribir la consulta.
  - Evita el uso de funciones incorporadas para la optimización de consultas.

**Problema 2: Diseño de sistema distribuido**
- **Descripción:** Diseña un sistema distribuido para una aplicación de alto tráfico. Considera la escalabilidad, la disponibilidad y la tolerancia a fallos en tu diseño.
- **Restricciones:**
  - Describe la arquitectura del sistema y las tecnologías que usarías.
  - Considera aspectos como la consistencia eventual y la replicación de datos.

**Problema 3: Algoritmo de aprendizaje automático**
- **Descripción:** Implementa un algoritmo de aprendizaje profundo, como una red neuronal convolucional (CNN) para la clasificación de imágenes.
- **Restricciones:**
  - Puedes usar cualquier marco de aprendizaje profundo (por ejemplo, TensorFlow, PyTorch).
  - Entrena el modelo en un conjunto de datos relevante y evalúa su precisión.

#### 2. **Entrevista 2:**

  - Justifica tus decisiones de diseño y explica cómo manejarías las consultas específicas de la aplicación.

**Problema 2: Optimización de algoritmo**

- **Restricciones:**

**Problema 3: Arquitectura de microservicios**

- **Restricciones:**
  - Describe la estructura de los microservicios y cómo se comunican entre sí.
  - Considera aspectos como la autenticación, la autorización y la monitorización.

 Describe las técnicas y herramientas específicas que utilizarías para asegurar la aplicación.

**Problema 2: Optimización de la arquitectura en la nube**

  - Describe cómo estructurarías la arquitectura para optimizar los costos y el rendimiento.

#### 4. **Entrevista 4:**

**Problema 1: Arquitectura de alta disponibilidad y tolerancia a fallos**

#### 5. **Entrevista 5:**

**Problema 1: Sistema de recomendación**
- **Descripción:** Diseña un sistema de recomendación que sugiera productos a los usuarios en función de su historial de compras y comportamiento en el sitio web.
- **Restricciones:**
  - Describe cómo recopilarías y procesarías los datos del usuario para generar recomendaciones precisas.
  - Detalla cómo evaluarías la efectividad del sistema de recomendación.

**Problema 2: Arquitectura de seguridad de aplicación web**
- **Descripción:** Diseña una arquitectura de seguridad para una aplicación web que maneja información sensible del usuario.
- **Restricciones:**
  - Describe cómo implementarías autenticación de dos factores y encriptación de extremo a extremo para los datos del usuario.
  - Considera técnicas como la prevención de ataques CSRF y XSS.

**Problema 3: Diseño de sistema de caché distribuida**
- **Descripción:** Diseña un sistema de caché distribuida para una aplicación web de alto tráfico.
- **Restricciones:**
  - Describe cómo distribuirías la caché y cómo manejarías la invalidación de datos en la caché.
  - Considera técnicas como caché por capas y estrategias para minimizar las colisiones de caché.

Estos problemas están diseñados para evaluar las habilidades técnicas y la capacidad de resolución de problemas de los candidatos en diferentes niveles, desde desarrolladores junior hasta desarrolladores senior. Las restricciones están diseñadas para fomentar la creatividad y la comprensión profunda de los conceptos, alentando a los candidatos a encontrar soluciones eficientes y efectivas.
